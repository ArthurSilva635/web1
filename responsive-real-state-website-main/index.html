Projetos.

Projeto psicologia

Além dos trabalhos desenvolvidos dentro de campo, acreditamos que a evolução e resultados positivos para nossos alunos se faz com dinâmicas em grupos e indivuais relacionadas a psicologia do esporte.

Projeto palestra

Acreditamos que exemplos positivos agregam na formação de nossos alunos, por isso ao longo do período de treinamento recebemos visitas de profissionais de diferentes áreas.

Projeto Core

Sabemos a importância de nossos alunos estarem bem físicamente durante os treinamentos, por isso desenvolvemos de forma programada e planejada para cada tipo de faixa etaria, treinamento de CORE e sessões de ALONGAMGENTO.

Projeto Leitura

Além do esporte acreditamos no poder da educação, sendo assim desenvolvemos com nossos alunos o PROJETO LEITURA, onde temos parte do tempo do treinamento em um dia específico destinado a escolher um livro para leitura.

Treinamento específico para goleiros.

Temos a turma mais indicada para você

INICIANTES

Treinamento específico de coordenação motora com os fundamentos básicos da posição.

INTERMEDIÁRIO

Treinamento englobando agilidade, aperfeiçoamento dos fundamentos técnicos desenvolvidos no nível anterior e inserção a algumas situações de jogo.

AVANÇADO

Inserção de treinamento de força e condicionamento físico específico, manutenção dos fundamentos técnicos adquiridos nos estágios anteriores. Foco mais amplo a exercícios situacionais, inserindo o jogo dentro do treinamento.

TURMA EXCLUSIVA DE ADULTOS

Desenvolvemos uma turma somente com goleiros adultos, onde são divididos dentro dela mesmo todos os estágios mencionados acima, contando com um professor exclusivo somente para os adultos.
